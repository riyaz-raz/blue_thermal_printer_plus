# blue_thermal_plus

A Flutter plugin for connecting to thermal printer via bluetooth (Android Only).

## Getting Started

### 1. Depend on it

Add this to your package's pubspec.yaml file:

```yaml
dependencies:
  blue_thermal_plus: ^any
```

### 2. Install it

You can install packages from the command line:

with Flutter:

```bash
$ flutter packages get
```

### 3. Import it

Now in your Dart code, you can use:

```dart
import 'package:blue_thermal_plus/blue_thermal_plus.dart';
```

## Note
Make sure to upgrade your project to compileSdkVersion 34 or above.
